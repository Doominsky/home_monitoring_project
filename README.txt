RO

Etape pentru utilizarea proiectului:

1. Urmariti acest videoclip pentru a seta Google API: https://www.youtube.com/watch?v=rGTIJ0yvlQA
2. Pe langa fisierul main.py de tip Python, va fi nevoie si de un fisier client_secret.json care va contine parametrii contului d-voastra de Google Drive
3. In codul de Python, linia 98 si 212, trebuie schimbata calea de salvare, cu o cale valida aferenta dispozitivului d-voastra de calcul 
4. Conectati dispozitivul cu Arduino la un port USB
5. Asigurati-va ca sistemul este conectat la internet prin Firebase (datele se vor schimba atat in website, cat si in baza de date)
6. Porniti script-ul
7. Introduceti un numar de minute pentru interogare (numar intreg)
8. Veti primi o copie Excel a interogarii in stocarea dispozitivului de calcul, si o copie in cadrul contului Google

EN

Steps for using the project:

1. Watch this video to set up the Google API: https://www.youtube.com/watch?v=rGTIJ0yvlQA
2. In addition to the Python main.py file, you will also need a client_secret.json file that will contain the parameters of your Google Drive account.
3. In the Python code, line 98 and 212, you must change the save path to a valid path for your computing device 
4. Connect your Arduino device to a USB port
5. Make sure the system is connected to the internet via Firebase (data will change both in the website and in the database)
6. Start the script
7. Enter a number of minutes for the query (integer)
8. You will receive an Excel copy of the query in your computing device storage, and a copy in your Google account
